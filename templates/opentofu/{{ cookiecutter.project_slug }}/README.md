## Quick Start

1. **Install tools** (mise handles OpenTofu, terraform-docs, tflint, etc.):
   ```bash
   mise install
   ```

2. **Apply infrastructure**:
   ```bash
   {%- if cookiecutter.subscription_id %}
   mise run apply
   {%- else %}
   # Set subscription_id first:
   export TF_VAR_subscription_id=YOUR_SUBSCRIPTION_ID
   mise run apply
   {%- endif %}
   ```

3. **View available tasks**:
   ```bash
   mise tasks
   ```
